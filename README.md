# PostgreSQL
PostgreSQL  Tutorial 

## Useful Commands

```bash
psql -U postgres
```

### Create Database

```bash
CREATE DATABASE <database name>;
```

### Connect to a database

```bash
psql -h localhost -p 5432 -U postgres <database name>

\l # list the databases
\c # connect to the database
```
